# "마이링크" (MyLink) 제품 요구사항 정의서 (PRD)

## 1. 프로젝트 개요
* **프로젝트명**: 마이링크 (MyLink)
* **목적**: 여러 소셜 미디어 플랫폼과 웹사이트 링크를 단일 페이지에 모아서 공유할 수 있는 나만의 프로필 링크 서비스 제공
* **대상 사용자**: 
  * 여러 채널(인스타그램, 유튜브, 블로그, 깃허브 등)을 운영하는 크리에이터 및 인플루언서
  * 개인 포트폴리오나 이력서를 하나의 링크로 공유하고 싶은 취업 준비생 및 전문가
  * 서비스나 상품 홍보 채널을 묶어서 제공해야 하는 소규모 비즈니스 운영자

## 2. 핵심 기능 목록

### 필수 기능 (MVP - Minimum Viable Product)
* [ ] **사용자 인증 (Auth)**: 회원가입, 로그인, 로그아웃 (이메일/비밀번호 기반)
* [ ] **프로필 관리**: 프로필 이미지 업로드, 닉네임, 간단한 자기소개 설정
* [ ] **링크 관리 (CRUD)**: 
  * 링크 추가 (제목, URL)
  * 링크 수정, 삭제
  * 링크 노출 여부(On/Off) 토글
  * 링크 노출 순서 변경 (Drag & Drop)
* [ ] **프로필 페이지 사이트 생성**: 사용자별 고유 URL 제공 (예: `mylink.com/username`) 및 설정된 링크 목록 렌더링 지원

### 선택 기능 (고도화/추가 기능)
* [ ] **소셜 로그인**: 구글, 깃허브, 카카오 등 SNS 원클릭 연동
* [ ] **디자인 커스터마이징**: 
  * 프로필 페이지 배경(단색, 그라데이션, 이미지) 설정
  * 버튼 스타일 (둥근 모서리, 강한 그림자 등), 테마 및 폰트 변경
* [ ] **링크 트래킹 요약 (Analytics)**: 각 링크별 누적 클릭 수 및 프로필 전체 방문자 수 제공
* [ ] **미디어/소셜 위젯**: 단순 텍스트 링크 외에 유튜브 영상 임베드, 스포티파이 음악 재생 위젯 등 제공

## 3. 기능 상세 설명

### 3.1 사용자 인증 및 계정 설정
* 사용자는 고유한 문자열인 '아이디(username)'를 기반으로 가입하며, 이것이 프로필 URL의 고유 식별자가 됩니다.
* 계정 단위의 설정 페이지를 통해 비밀번호 변경, 계정 탈퇴 등의 기본 보안 기능을 지원합니다.

### 3.2 대시보드 (관리자 페이지)
* 로그인 후 진입하는 메인 작업 공간입니다.
* 화면 좌측(또는 메인 영역)에는 링크를 추가/수정하는 관리 UI를, 화면 우측(또는 모달 영역)에는 데스크탑 및 모바일 뷰어 모드의 **'실시간 미리보기 뷰어(Preview)'**를 배치하여 수정 사항을 직관적으로 확인할 수 있게 합니다.

### 3.3 링크 설정 디테일
* **썸네일 및 아이콘**: 각 링크 버튼 좌측에 들어갈 작은 아이콘(소셜 로고 등)이나 이미지를 설정할 수 있습니다.
* **강조 효과(Animation)**: 특정 중요한 링크를 방문자가 클릭하도록 유도하기 위해 흔들림(Shake), 반짝임(Pulse) 등의 간단한 애니메이션 추가 기능을 제공합니다.

### 3.4 공개 프로필 페이지
* 모든 디바이스에서 최적화된 **반응형 웹 UI(주로 모바일에 최적화)**로 제공됩니다.
* **동적 SEO 태그 적용**: 사용자가 자신의 링크를 메신저(카카오톡, 슬랙 등)에 공유할 때 예쁜 썸네일과 제목이 렌더링 되도록 `Open Graph(OG)` 태그를 동적으로 생성합니다.

## 4. 제안하는 개선점 (AI 특별 제안)
비슷한 수많은 대체 서비스들 사이에서 '마이링크'만의 특별한 장점이 될 수 있는 추가 아이디어를 제안합니다.

1. **자동 QR 코드 생성기**: 사용자의 프로필 URL과 연동되는 고유 QR 코드를 대시보드에서 바로 이미지로 다운타운받을 수 있게 하여 오프라인(명함, 포스터) 과 연결짓는 편의성을 높입니다.
2. **일정 스케줄링 (기간 한정 링크)**: 특정 이벤트용 링크를 올릴 때, '이 링크는 OOOO년 OO월 OO일 오전까지만 클릭 가능함' 같이 설정하면 시간이 지난 후 자동으로 숨겨지는 기능입니다.
3. **비밀번호 기반 시크릿 링크**: 특정 링크에 비밀번호 자물쇠를 걸어두어, 비밀번호를 입력한 사용자에게만 URL을 다이렉트 해주는 기능입니다 (자료 배포, 포트폴리오 열람 등에 유용함).
4. **연락처 (vCard) 다운로드 위젯**: 단순 페이지 뷰뿐 아니라, 방문자가 버튼 하나로 크리에이터나 비즈니스의 이메일, 전화번호 등 연락처를 휴대폰 주소록에 즉시 저장할 수 있게 돕는 기능입니다.

## 5. 기술 스택 및 기본 정책
1. **프론트엔드**: Next.js 기반, **Shadcn/ui** 디자인 시스템 적용 (깔끔하고 모던한 UI 추구)
2. **백엔드/DB**: **Firebase** 활용 (Firebase Auth, Firestore, Firebase Storage)
3. **서비스 정책**: 프리미엄 제약 없이 모든 기능을 **전면 무료**로 제공 (사용자 확보 및 만족도 제고)
