# 마이링크 (MyLink) - PC 기준 와이어프레임

PC 화면(데스크탑 뷰)을 기준으로 한 와이어프레임(구조 설계도)입니다. 모든 핵심 UI는 가로로 넓은 모니터 환경에서 사용자가 쾌적하게 작업할 수 있도록 공간을 효율적으로 배치합니다.

---

## 1. 로그인/회원가입 페이지
**설명**: 서비스에 최초로 진입하여 인증을 거치는 화면입니다. 구글 로그인(MVP)을 최우선으로 배치합니다.

```text
+-------------------------------------------------------------+
|  [마이링크 로고]                                            |
|                                                             |
|          +---------------------------------------+          |
|          |         나만의 링크 만들기            |          |
|          |         모든 링크를 한 곳에서!        |          |
|          |                                       |          |
|          |  [ G 구글(Google) 계정으로 시작하기 ] |          |
|          |                                       |          |
|          |  이미 계정이 있으신가요? 로그인       |          |
|          +---------------------------------------+          |
|                                                             |
+-------------------------------------------------------------+
```

---

## 2. 관리자 대시보드 (Split-View 구조)
**설명**: 로그인 직후 진입하는 핵심 레이아웃입니다. 넓은 PC 화면의 이점을 살려 좌측은 스크롤 가능한 [관리/입력 영역]을, 우측은 실시간 [프로필 화면 뷰어]로 양분하여 구성(Split-view)합니다.

```text
+-------------------------------------------------------------------------+
| [마이링크 로고]   (링크 탭)   (디자인 탭)   (통계 탭)     [내 주소 복사]|
+-------------------------------------------------------------------------+
|                                  |                                      |
|  [ + 새로운 링크 추가하기 ]      |   +------------------------------+   |
|                                  |   |                              |   |
|  = [Github아이콘] 내 깃허브 (On) |   |        [프로필 이미지]       |   |
|    URL: github.com/poatan2       |   |           Nickname           |   |
|   [수정] [삭제] [통계: 50클릭]   |   |      "안녕하세요! 개발자..." |   |
|  ------------------------------  |   |                              |   |
|                                  |   |  [ 내 깃허브 바로가기      ] |   |
|  = [Youtube아이콘] 내 유튜브 (On)|   |                              |   |
|    URL: youtube.com/...          |   |  [ 내 유튜브(구독)         ] |   |
|   [수정] [삭제] [통계: 12클릭]   |   |                              |   |
|  ------------------------------  |   |  [ 블로그 글 보러가기      ] |   |
|                                  |   |                              |   |
|  (드래그 앤 드롭으로 순서 변경)  |   |    * 실시간 데스크탑 뷰어    |   |
|                                  |   +------------------------------+   |
+-------------------------------------------------------------------------+
```

## 3. 통계(Analytics) 페이지 (Level 2)
**설명**: Firebase에서 취합한 데이터를 기반으로 날짜별 트렌드 차트 및 누적 데이터를 보여주는 대시보드입니다.

```text
+-------------------------------------------------------------------------+
| [로고]   (링크)   (디자인)   (*통계*)                   [내 주소 복사]|
+-------------------------------------------------------------------------+
|                                                                         |
|  ## 내 프로필 분석 요약                                                 |
|  +---------------------------+  +---------------------------+           |
|  | 총 방문자 수 (View)       |  | 총 링크 클릭 수 (Click)   |           |
|  |     1,245 명              |  |      342 회               |           |
|  +---------------------------+  +---------------------------+           |
|                                                                         |
|  ## 방문자 추이 (최근 7일)                                              |
|    ^                                                                    |
| 100|       * (100)                                                      |
|  50|      / \       *----* (60)                                         |
|  20| *---*   \     /      \                                             |
|   0+----------*---*--------*------------------------------> (일자별)    |
|      1일  2일  3일  4일  5일  6일  7일                                  |
|                                                                         |
|  ## 개별 링크 클릭 순위                                                 |
|  1. 내 깃허브 바로가기 .................... 200회                       |
|  2. 블로그 글 보러가기 .................... 100회                       |
|  3. 내 유튜브(구독) ....................... 42회                        |
|                                                                         |
+-------------------------------------------------------------------------+
```

---

## 4. 퍼블릭 프로필 페이지 (방문자가 보는 실제 화면)
